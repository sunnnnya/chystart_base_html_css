表单的基本控件

  • label标签：点击范围的扩大
    • <label for="female">Xxx</lable>
  • 文本输入框：<input type="text" />
    • name 属性：指定控件名称， 提交的 key
    • value 属性：指定默认值
    • maxlength 属性：指定可输入的文本最大长度
    • disabled 属性：禁用表单控件
  • 密码输入框：<input type="password" />
    • name 属性：指定控件名称， 提交的 key
    • maxlength 属性：指定可输入的文本最大长度
  • 隐藏输入框：<input type="hidden" />
    • name 属性*：指定控件名称，提交的 key
    • value 属性*：选定的值， 提交的 value
  • 单选框：<input type="radio" />
    • name 属性*：指定控件名称，必须相同才能只选择一个
    • value 属性*：选定的值
    • checked 属性：默认选中
  • 多选框：<input type="checkbox" />
    • name 属性*：指定控件名称
    • value 属性*：选定的值
    • checked 属性：默认选中
  • 文本域：<textarea name="Xxx" cols="Xxx" rows="Xxx"></textarea >
    • rows：行
    • cols：列
  • 下拉框：<select name="Xxx">
                            <option value="Xxx"> </option >
                            <option value="Xxx"> </option >
                    </select >
    • name：属性，表示传递到后端的key的名称
    • value：属性，表示每个选项对应的值
    • selected：属性，表示下拉框初始展示的值
  • 按钮：<button></button > 或 <input type="submit" />
    • 提交按钮：<button type = "submit"> </button>
    • 重置按钮：<button type = "reset"> </button>
    • 普通按钮：<button type = "button"> </button>
