# 金香烤鸡售卖网站

一个现代化、响应式的烤鸡售卖网站，包含产品展示、购物车和结账功能。

## 功能特点

- 🍗 **产品展示**：8种不同口味的烤鸡产品
- 🛒 **购物车系统**：添加/删除商品，修改数量，实时计算总价
- 💰 **结账流程**：订单信息填写，订单确认
- 📱 **响应式设计**：适配桌面、平板和手机设备
- 🎨 **现代化界面**：温暖、开胃的配色方案，流畅的动画效果
- 💾 **本地存储**：购物车数据保存在浏览器本地存储中

## 快速开始

1. 直接使用浏览器打开 `index.html` 文件
2. 或者使用本地服务器运行：
   ```bash
   # 使用Python启动简单HTTP服务器
   python -m http.server 8000

   # 然后访问 http://localhost:8000
   ```

## 使用方法

### 浏览产品
- 浏览8种不同口味的烤鸡产品
- 查看产品描述、价格和特惠信息
- 使用数量选择器调整购买数量

### 购物车操作
- 点击"加入购物车"添加商品
- 点击右上角购物车图标查看购物车
- 在购物车中调整商品数量或删除商品
- 实时显示购物车总价

### 结账流程
1. 点击"去结账"按钮
2. 填写收货信息（姓名、电话、地址）
3. 查看订单摘要
4. 点击"确认下单"提交订单
